# vue-tutorselectsystem :raising_hand:
## Background
更加智能化、快捷的学生导师互选系统。<br/>
在传统学生导师互选简单操作流程外，导师可以提前加入心仪学生；可以自由设置录取学生的标准，系统按照导师给出的标准计算出符合要求的学生，符合要求的学生才可进行选择，学生在选择时不符合要求及时告知，以不影响学生对其他老师的选择。添加了对学生毕业设计方向的管理，指导学生更加方便。

## Install
Node.js v12.16.1
Visual Studio Code
